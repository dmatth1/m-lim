Top Remaining Visual Differences
---------------------------------
1. WAVEFORM DISPLAY (primary driver, ~26% RMSE):
   Reference shows an active audio session — blue/purple waveform fill with
   yellow/red gain reduction overlay. M-LIM shows an empty dark waveform
   (expected for idle standalone). This is ~60% of the plugin area. This
   difference is structural and cannot be resolved without feeding live audio.

2. LEFT LEVEL METERS (highest per-region RMSE, ~30%):
   Despite task 257's LED-segment implementation, the meter bars still read
   near-zero (idle state). Pro-L 2 reference shows active signal with colored
   green→yellow→red segments. The segment count, spacing, and color gradient
   match may need further tuning. The scale labels and gutter width differ.

3. COLOR PALETTE / OVERALL BRIGHTNESS (~25% RMSE throughout):
   Pro-L 2 uses a lighter blue-gray theme with more contrast. M-LIM's theme is
   darker. The algorithm selector buttons, control strip background, and right-
   panel LUFS readout area all contribute. The output slider area appears
   significantly different in coloring.

4. LOUDNESS PANEL RIGHT SIDE (~26%):
   The LUFS readout section in M-LIM is darker with different layout spacing
   vs. Pro-L 2's brighter gray panel. The Momentary/Short-Term/Integrated/Range
   row heights and font sizes appear to differ.

5. BOTTOM STATUS BAR (included in control strip, ~22%):
   The status bar row (True Peak Limiting label with green underline, TP/Pause/
   Loudness buttons at right) uses different colors/styling. The TP indicator
   and pause button positions have been fixed per task 263, but overall bar
   color scheme still differs.

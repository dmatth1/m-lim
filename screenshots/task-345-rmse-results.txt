Acceptance Criteria
-------------------
Full image ≤ task-343 (22.05%):    FAIL — 22.99% > 22.05%
Waveform ≤ task-343 (20.20%):      FAIL — 21.97% > 20.20%

Decision
--------
Task-341 change (0xff506090 → 0xff4A4F6B) caused a regression across all regions.
The task-341 analysis assumed reference bottom was ~#464B67 but actual reference
composite pixel at y=380 is #7A809B (lighter). The pre-existing value 0xff506090
was empirically better matched to the reference composite appearance.

Action: Reverted displayGradientBottom to 0xff506090 (pre-task-341 value).
The current codebase maintains the task-343 best of 22.05% full-image RMSE.
